# Battleship Game

A networked Battleship game implementation using Java sockets for client-server communication.

## Project Structure

- **server/** - Server-side implementation
  - `BattleshipServer.java` - Main server class
  - `ClientHandler.java` - Handles individual client connections
  - `GameState.java` - Manages game state

- **client/** - Client-side implementation
  - `BattleshipClient.java` - Main client class
  - `ServerListener.java` - Listens for server messages

- **common/** - Shared code
  - `Protocol.java` - Protocol definitions
